Fork updates (Marchand):

  • Pointer → water ripples: correct device-pixel ray hit (so clicks make waves)
  • WGSL #include via Vite (no raw imports on entry shaders)
  • Pool / world height scaling aligned in shaders
  • Gravity (G) on by default; optional buoyant sphere physics
  • Help panel: collapsed by default, full scroll, menu toggle (top-right)
  • GitHub Pages deploy with base /water/

This demo requires a browser that supports WebGPU. If you can't run the demo, you can still see the original WebGL implementation at madebyevan.com/webgl-water/.

Interactions:

  • Draw on the water to make ripples
  • Drag the background or Right-click to rotate the camera
  • Use the Mouse wheel to zoom in and out
  • Press SPACEBAR to pause and unpause
  • Drag the sphere to move it around
  • Press the L key to set the light direction
  • Press the G key to toggle gravity (on by default)
  • Click the menu icon (top-right) to show or hide this panel

Features:

  • Raytraced reflections and refractions
  • Analytic ambient occlusion
  • Heightfield water simulation
  • Adjustable object density (physics)
  • Soft shadows
  • Real-time Caustics
